# Información
Se ha hecho un CRUD usando express, node y mongodb. Para la parte visual se ha usado ejs, router, controller y un servicio para redirigir las llamadas desde el front al back.
Las vistas se han dividido en componentes para el header, footer y los formularios. 

# Validaciones
Se ha validado tanto los formularios del front mediante el "validator" de Boostrap, así como en el back en el modelo de moongose.

# Instalación
Descargar repositorio de github, entrar en la carpeta y lanzar "npm install"

# Ejecución
1 - Abrir cmd y correr "mongod.exe"

2 - En vsCode ejecutar "npm run nodemon.start"

# Test
Abrir Postman, importar la colección, añadida en la carpeta "postman" del proyecto. Lanzar las queries. Será necesario lanzar el POST primero para crear usuarios.

# Aplicación
Abrir la aplicación en nuestro navegador con "http://localhost:3000"
